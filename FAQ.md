# 常见问题 (FAQ)

## 安装与运行

### Q: 如何安装应用程序？

**A:** 无需安装，直接运行可执行文件即可。
- Windows: 双击 `经典云网盘.exe`
- Linux: 运行 `./经典云网盘`（需要执行权限）
- macOS: 双击 `经典云网盘.app`

### Q: 需要安装 Python 吗？

**A:** 不需要。应用程序已包含所有必要的依赖，无需安装 Python 环境。

### Q: 首次运行需要什么？

**A:** 
- 网络连接（用于登录和同步数据）
- 有效的账号（如果没有，可以在应用程序中注册）

## 功能使用

### Q: 如何上传文件？

**A:** 
1. 点击工具栏"上传文件"按钮
2. 选择要上传的文件（支持多选）
3. 文件会自动上传到当前文件夹

### Q: 如何下载文件？

**A:** 
1. 在文件列表中选中文件
2. 点击工具栏"下载"按钮
3. 或双击文件进行下载
4. 选择保存位置

### Q: 如何分享文件？

**A:** 
1. 选中要分享的文件或文件夹
2. 点击工具栏"分享"按钮
3. 复制生成的分享链接
4. 分享给其他人

### Q: 如何搜索分享链接？

**A:** 
1. 点击工具栏"搜索分享"按钮
2. 输入分享链接或分享码
3. 支持自动识别URL中的分享码
4. 查看分享信息并下载文件

### Q: 下载次数在哪里显示？

**A:** 下载次数显示在文件列表的"下载次数"列中，与官网显示一致。

## 问题排查

### Q: 无法登录怎么办？

**A:** 请检查：
1. 网络连接是否正常
2. 用户名和密码是否正确
3. API地址是否正确（默认：https://gta5fuzhup.cn）
4. 查看日志文件获取详细错误信息

### Q: 上传失败怎么办？

**A:** 请检查：
1. 文件大小是否超过限制
2. 存储空间是否充足
3. 网络连接是否稳定
4. 查看日志文件获取详细错误信息

### Q: 下载失败怎么办？

**A:** 请检查：
1. 网络连接是否正常
2. 下载路径是否有写入权限
3. 磁盘空间是否充足
4. 查看日志文件获取详细错误信息

### Q: 应用程序无法启动？

**A:** 请检查：
1. 系统要求是否满足
2. 查看日志文件获取错误信息
3. 尝试以管理员权限运行（Windows）
4. 检查防病毒软件是否阻止运行

### Q: 如何查看日志文件？

**A:** 日志文件位置：
- Windows: `C:\Users\用户名\.classic_cloud\logs\app_YYYYMMDD.log`
- Linux/macOS: `~/.classic_cloud/logs/app_YYYYMMDD.log`

## 配置与设置

### Q: 配置文件在哪里？

**A:** 配置文件位置：
- Windows: `C:\Users\用户名\.classic_cloud\config.json`
- Linux/macOS: `~/.classic_cloud/config.json`

### Q: 如何修改下载路径？

**A:** 
1. 在应用程序设置中修改
2. 或直接编辑配置文件中的 `download_path` 字段

### Q: 如何清除登录信息？

**A:** 
1. 退出登录
2. 或删除配置文件中的 `token` 字段

## 平台相关

### Q: 支持哪些操作系统？

**A:** 
- Windows 7/8/10/11
- Linux (Ubuntu, Debian, CentOS等)
- macOS 10.12+

### Q: 支持 Apple Silicon (M1/M2/M3) 吗？

**A:** 是的，支持 Intel 和 Apple Silicon 芯片。

### Q: 支持 32位系统吗？

**A:** Windows 支持 32位和 64位，Linux 和 macOS 主要支持 64位。

## 数据与隐私

### Q: 数据存储在哪里？

**A:** 
- 文件存储在云端服务器
- 配置文件存储在本地用户目录
- 日志文件存储在本地用户目录

### Q: 数据安全吗？

**A:** 
- 所有数据传输使用 HTTPS 加密
- Token 自动加密存储
- 配置文件仅所有者可访问
- 不收集任何用户隐私信息

### Q: 如何备份数据？

**A:** 
- 文件数据存储在云端，自动备份
- 配置文件可以手动备份
